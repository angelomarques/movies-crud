# 🚀 Projeto Filmes CRUD

Este é um projeto frontend utilizando **Vite**. Siga os passos abaixo para rodá-lo localmente.

---

## ⚙️ Pré-requisitos

- [Node.js](https://nodejs.org) instalado (versão recomendada: 18+)
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/) instalado

---

## 📁 1. Crie o arquivo `.env`

Crie um arquivo `.env` na raiz do projeto com as variáveis de ambiente necessárias.  
Exemplo:

```env
VITE_API_URL=http://localhost:3000
````

---

## 📦 2. Instale as dependências

Com **npm**:

```bash
npm install
```

Ou com **yarn**:

```bash
yarn
```

---

## ▶️ 3. Rode o projeto

Com **npm**:

```bash
npm run dev
```

Ou com **yarn**:

```bash
yarn dev
```

---

## 🌐 4. Acesse no navegador

Abra o navegador e vá para:

```
http://localhost:5173
```

---

## 📝 Observações

* Certifique-se de que o backend (caso necessário) esteja rodando corretamente.
* O Vite usa a porta `5173` por padrão. Se estiver ocupada, ele escolherá outra automaticamente.
